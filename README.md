# genai_playbooks

A personal collection of useful Markdown files, instructions, and templates for working with LLMs, Codex, ChatGPT, and GenAI automation.

The goal of this repository is to collect practical playbooks and reusable files that improve how AI assistants behave across real workflows: coding, security architecture, documentation, automation, prompt reuse, and engineering collaboration.


## Playbook Library

| File | Purpose | Reference |
| --- | --- | --- |
| [AGENTS.md](AGENTS.md) | Solves the repeated-context problem for Codex by giving it reusable personal guidance for security architecture work, audience-aware communication, coding behavior, quality standards, and workplace-ready outputs. | [docs/README_AGENTS.md](docs/README_AGENTS.md) |
